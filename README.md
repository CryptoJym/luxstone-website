## Features

- Executive Dashboard with key metrics and KPIs
- Leadership analysis and team structure
- Investment strategy and portfolio breakdown
- Financial analysis and projections
- Competitive positioning assessment
- Strategic predictions for 2025-2026
- Comprehensive citations and source verification

## Technology Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Chart.js for data visualization
- Vercel for hosting

## Deployment

The website is automatically deployed to Vercel when changes are pushed to the main branch.

## Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/CryptoJym/luxstone-website.git
   ```

2. Open `index.html` in your browser

3. Make changes and push to GitHub for automatic deployment

## Structure

- `index.html` - Main HTML file with all content
- `style.css` - Styling and responsive design
- `app.js` - JavaScript for interactivity and charts
- `vercel.json` - Vercel deployment configuration
